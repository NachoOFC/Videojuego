OPCIÓN 1 - LA MÁS FÁCIL (Windows):
  1. Haz doble clic en: run_server.bat
  2. Se abre automáticamente en tu navegador
  3. Presiona ENTER para comenzar
  4. ¡A jugar! 🎮

OPCIÓN 2 - Por terminal:
  PowerShell: python server.py
  Terminal: python -m http.server 8000
  Luego abre: http://localhost:8000

OPCIÓN 3 - Con npm:
  npm install
  npm start
